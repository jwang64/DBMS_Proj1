HOW TO COMPILE:
- Use the command:
      $ javac -d classes src/*.java
      
HOW TO EXECUTE:
- Use the command:
      $ java -cp classes MovieDB

HOW TO CREATE A JAVADOC:
- Use the command:
      $ javadoc -d doc -sourcepath src -subpackages
